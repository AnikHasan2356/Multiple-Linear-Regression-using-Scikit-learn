# Multiple Linear Regression in Machine Learning using Scikit-learn

This repository demonstrates the implementation of Multiple Linear Regression using Scikit-learn in Python. It explains how a machine learning model can use multiple input features to predict a continuous output variable efficiently using built-in libraries.

## Topics Covered

- Introduction to Multiple Linear Regression
- Difference between Simple and Multiple Linear Regression
- Data preprocessing for regression models
- Splitting dataset into train and test sets
- Model training using Scikit-learn
- Making predictions with trained model
- Model evaluation (R² score, MAE, MSE, RMSE)
- Understanding coefficients and intercept
- Feature impact interpretation
